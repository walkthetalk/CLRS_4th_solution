\startEXERCISE\DIFFICULT
在本節給出的第一個卡片排序算法中，
會先按最高位數字進行排序，
要給 $d$ 位十進制數排序，
最壞情況下需要多少輪？
最壞情況下，操作員需要記錄多少堆卡片？
\stopEXERCISE

\startANSWER
算法執行過程爲：
\startCLRSCODE
- // 按第一位分組
	- // 將第一位爲 0 的分組
		- // 將第二位爲 0 的分組
      			// 。。。
    		- // 將第二位爲 1 的分組
      			// 。。。
    		- // 將第二位爲 9 的分組
      			// 。。。
  	- // 將第一位爲 1 的分組
    		// 。。。
    	// 。。。
  	- // 將第一位爲 9 的分組
    		// 。。。
\stopCLRSCODE

可以看出算法爲指數級的。每位數的範圍爲 $0$ 到 $k$，
需要時間爲 $\Theta(k^d)$，即 $k^d$ 輪排序，
空間則爲 $\Theta(nk)$，即需要 $nk$ 堆卡片。
\stopANSWER
