\startEXERCISE
有兩個 $n$ 位二進制整數 $a$ 和 $b$，
存儲在兩個元素數目為 $n$ 的數列 $A[0:n-1]$ 和 $B[0:n-1]$ 中，
數列中的所有元素均為 $0$ 或 $1$，
$a = \sum_{i=0}^{n-1}A[i]\cdot 2^i$，
$b = \sum_{i=0}^{n-1}B[i]\cdot 2^i$。
數列 $C[0:n]$ 用於存儲兩數之和 $c=a+b$，
$c = \sum_{i=0}^{n}C[i]\cdot 2^i$。
請給出例程 \ALGO{ADD-BINARY-INTEGERS}，
其輸入為數列 $A$ 和 $B$，長度均爲 $n$，
返回的數列 $C$ 中存儲兩數之和。
\stopEXERCISE
\startANSWER

\startCLRSCODE
for i = 1 to n + 1
	C[i] = 0
for i = 1 to n
	if A[i] == 1 and B[i] == 1
		C[i+1] = 1
	else if A[i] == 1 or B[i] == 1
		if C[i] == 1
			C[i] = 0
			C[i+1] = 1
		else
			C[i] = 1
\stopCLRSCODE

\stopANSWER
